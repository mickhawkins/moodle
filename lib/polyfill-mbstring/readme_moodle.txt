Instructions to import polyfill-mbstring into Moodle:

1/ Download from https://github.com/symfony/polyfill-mbstring/releases

2/ Copy the LICENSE file into the lib/polyfill-mbstring folder

3/ Copy the following into the lib/polyfill-mbstring/src folder:
   - Resources folder
   - bootstrap.php
   - Mbstring.php
